# Introduction to HTML

HTML stands for **HyperText Markup Language**. It is the basic language used to create web pages. HTML tells the web browser how to display text, images, links, and other content on a webpage.

HTML is not a programming language; it is a **markup language** that uses tags to structure content.

## Why HTML is Used

- To create web pages  
- To structure text (headings, paragraphs)  
- To add images, links, tables, and forms  
- To build the basic layout of websites  

## Features of HTML

- Easy to learn and use  
- Uses simple tags and elements  
- Supported by all web browsers  
- Platform independent

## Important HTML Tags

- `<html>` – Root of the HTML document  
- `<head>` – Contains page information  
- `<title>` – Shows title in browser tab  
- `<body>` – Contains visible content  
- `<h1>` to `<h6>` – Headings  
- `<p>` – Paragraph  
- `<br>` – Line break  

## Advantages of HTML

- Simple and easy to understand  
- Free and open source  
- Works with CSS and JavaScript  
- Supported by all devices  

## Conclusion

HTML is the foundation of web development. Every website uses HTML to display content. Learning HTML is the first step to becoming a web developer.
